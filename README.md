# Minesweeper (Python + Pygame)

마우스로 플레이하는 python 지뢰찾기 게임.

## 실행 방법

```bash
pip install -r requirement.txt
python main.py
```

> **참고:** Python 3.14 환경에서는 `pygame` 대신 `pygame-ce`(community edition)를 사용해야 합니다.
> `pygame 2.6.1`은 Python 3.14에서 font 모듈이 동작하지 않습니다.

---

## 파일 구조

```
minesweeper/
├── main.py        # 진입점, 게임 루프, 마우스 이벤트 처리, 승리 오버레이
├── board.py       # 게임 로직 (지뢰 배치, 셀 상태, 플러드필, 타이머)
├── renderer.py    # Pygame 렌더링 (그리드, 셀, 상단바, 타이머 표시)
├── records.py     # 순위 기록 로드/저장 (JSON)
└── records.json   # 난이도별 상위 5개 기록 (자동 생성)
```

## 구현 완료 기능

### 게임 로직 (`board.py`)
- [x] `Cell` dataclass — `has_mine`, `is_revealed`, `is_flagged`, `adjacent_mines`
- [x] 첫 클릭 안전 — 클릭 셀 ±1 범위(3×3) 제외 후 지뢰 배치
- [x] 좌클릭 열기 (`reveal`) — 지뢰/숫자/빈 셀 분기 처리
- [x] 빈 셀 자동 확장 — BFS 플러드필 (재귀 대신 deque 사용, 스택 오버플로우 방지)
- [x] 우클릭 깃발 토글 (`toggle_flag`)
- [x] 코드 클릭 (`chord`) — 공개된 숫자 셀에서 좌+우 동시 클릭 시 이웃 자동 공개
- [x] 패배 판정 — 지뢰 클릭 시 모든 지뢰 공개, `game_state = "lost"`
- [x] 승리 판정 — 지뢰 아닌 모든 셀 공개 시 `game_state = "won"`
- [x] 게임 리셋 (`reset`)

### 렌더링 (`renderer.py`)
- [x] 상단바 — 남은 지뢰 카운터(3자리 빨간 숫자), 리셋 버튼
- [x] 미공개 셀 — 회색 배경 + 3D 베벨 효과 (흰 테두리 상/좌, 어두운 테두리 하/우)
- [x] 공개 셀 — 밝은 회색 배경 + 숫자 (1~8 고유 색상)
- [x] 깃발 아이콘 — 빨간 삼각형 + 수직 막대 + 받침
- [x] 지뢰 아이콘 — 검은 원 + 방사형 선 8개 + 하이라이트
- [x] 트리거된 지뢰 — 빨간 배경으로 구분
- [x] 리셋 버튼 상태 표시 — 진행 중 `:)` / 승리 `W` / 패배 `X`

### 게임 루프 (`main.py`)
- [x] 360×420 창 (9×9 그리드 + 상단바 60px)
- [x] 좌클릭 열기 / 우클릭 깃발
- [x] 좌+우 동시 클릭 — 코드 클릭 (`chord`) 발동
- [x] 리셋 버튼 클릭 처리
- [x] 게임 종료 후 추가 클릭 차단

## 환경

- Python 3.14.3
- pygame-ce 2.5.7 (SDL 2.32.10)

## 기능 개선 예정 사항
- [x] 난이도 선택 (초급, 중급, 고급) : 난이도가 높아지면 패널의 크기가 커지고, 마인의 수가 많아짐
- [x] 시간 측정 기능 : 첫 클릭부터 게임 종료까지 상단바에 초 단위로 표시
- [x] 난이도별 순위 기록 기능 : 승리 시 상위 5개 기록 저장 및 오버레이 표시, 난이도 선택 화면에서 최고 기록 확인 가능
- [x] 순위 조회 기능 : 상단바 `기` 버튼 클릭 시 난이도별 탭 순위 오버레이 표시 (게임 중/종료 후 언제든 조회 가능)
